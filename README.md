## INSTALLATION
```bash
$ npm init
$ npm install express
$ npm install nodemon
$ npm install body-parser
$ npm install jsonwebtoken
$ npm install mysql
$ npm install morgan
$ npm install dotenv
$ npm install multer
$ npm install cors
```

### START THE PROJECT
```bash
$ npm start
```
